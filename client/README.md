# Avasar Patra Client

Frontend app for Avasar Patra (React + Vite + Tailwind).

## Run Frontend

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Demo Flow (Hackathon)

1. In the server folder, run demo bootstrap:

```bash
npm run demo:setup
```

2. Start backend:

```bash
npm run dev
```

3. Start frontend (this folder):

```bash
npm run dev
```

4. Login using demo credentials:
- Vendor: `demo.vendor@avasarpatra.com` / `DemoPass123`
- Admin: `admin@avasarpatra.com` / `AdminPass123`

5. Open Boardroom Brief page in app navigation and run Guided Demo Mode.

## Key Demo Pages

- `/dashboard` - personalized tender feed with explainability
- `/brief` - boardroom command center (snapshot, simulation, PDF export)
- `/operations` - assignment and readiness workspace
- `/vault` - document status and expiry management
